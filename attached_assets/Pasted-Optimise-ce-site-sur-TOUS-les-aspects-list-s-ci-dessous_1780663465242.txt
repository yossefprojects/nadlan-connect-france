⚙️ 6. TECHNIQUE

  X-Content-Type-Options: nosniff
  Referrer-Policy: strict-origin-when-cross-origin
  Content-Security-Policy de base
- Vérifie que le site fonctionne sans JavaScript activé (fallback ou message d'avertissement)
